## Screenshot Tips

### Using Browser DevTools
1. Press `F12` to open DevTools
2. Click device icon for mobile view
3. Use screenshot feature in DevTools

### Using Snipping Tool
- Windows: `Win + Shift + S`
- Mac: `Cmd + Shift + 4`
- Linux: `Shift + Print Screen`
